# 기업가정신 (Entrepreneurship)

> "기업가정신은 현재와 미래 변화를 관찰하고 기회를 포착하며 아이디어를 생성한 뒤,  
> 사업모델을 설계하는 것을 시작으로 이를 통해 직업과 직무를 창조하여 보급하거나  
> 자원을 모아 사업을 시작하거나 사회적 공익을 추구하는 **태도, 지식, 능력**"

---

## 기업가정신의 역사

| 시대 | 인물 | 정의 |
|------|------|------|
| 18세기 | 리차드 칸틸론 | 불확실성을 감수하고 수익을 얻는 사람 ('risk taker') |
| 19세기 | 장 밥티스트 세이 | 자원을 조합해 부가가치를 창출하는 사람 ('value creator') |
| 20세기 초 | **슘페터** | 창조적 파괴를 통해 경제 혁신을 일으키는 주체 |
| 20세기 중반 | **피터 드러커** | 기업가는 기회를 찾고 체계적으로 실행하는 사람 |
| 21세기 | 스티브 블랭크 | 고객 개발 이론: 기업가는 '문제를 검증하고 학습하는 사람' |

---

## 현대적 확장 3가지

| 유형 | 내용 |
|------|------|
| **인트라프러너(Intrapreneur)** | 조직 내에서 혁신을 추진하는 '행동하는 몽상가' |
| **사회적 기업가정신** | 이익보다 사회문제 해결 집중, 지속가능한 수익 창출 |
| **지속가능한 기업가정신** | 환경·윤리·지역사회까지 고려하는 확장된 관점 |

---

## 기업가 분류 4가지

| 유형 | 특징 |
|------|------|
| 혁신형 (Innovating) | 새로운 상품·방식·시장을 발견 |
| 모방형 (Imitating) | 혁신형의 성공적 혁신을 따라하고 채택 |
| 파비안형 (Fabian) | 변화에 매우 신중하고 회의적, 실험하지 않음 |
| 드론형 (Drone) | 수익이 감소해도 생산 공식을 바꾸지 않음 |

---

## 기업가정신 5가지 마인드셋

> **기업가 마인드셋 = 스타트업의 마인드셋 = 디자인씽킹과 거의 흡사**

| 특성 | 내용 |
|------|------|
| 호기심 | 기존 규범에 의문을 품고 새로운 가능성을 탐색 |
| 실행력 | 불확실한 상황 속에서도 우선 시도하고 학습 |
| 실패의 수용 | 실패를 피하지 않고 반복 학습의 과정으로 수용 |
| 자기 동기화 | 외부 보상보다 내면의 문제 의식에서 출발 |
| 비전과 가치 중심 | 단기 수익보다 세상을 더 좋게 만드는 방향 설정 |

---

## 슘페터의 창조적 파괴 — 3가지 통찰

1. 혁신은 **이미 있는 상품의 품질이나 서비스를 개선**하는 것도 포함
2. 혁신은 **이미 있는 상품을 새로운 생산방식으로 만드는 것**도 포함
3. 혁신은 **이미 있는 모든 것 위에 새로운 시장을 개척**하는 것도 포함

## 기업가정신 교육의 효과 (연구 데이터)

- 기업가정신 교육 받은 졸업자의 **27.2%**가 비즈니스를 수행 (vs 미교육자 9%)
- 교육 받은 기업가는 **5배 이상의 높은 매출 신장률**
- **3배 이상의 신제품 개발력**
- 평균 보유 자산: **28만$ vs 17만$** (62% 이상 높음)

## 이스라엘의 후츠파(Chutzpah) 정신

> "뻔뻔함"이 만드는 파괴적 혁신 — **무례함이 아닌, 본질을 찌르는 당돌함**

### 후츠파 7가지 요소

| 요소 | 내용 |
|------|------|
| Informality (형식 타파) | 격식보다 솔직한 표현, 권위보다 본질 |
| Questioning Authority (질문의 권리) | 상사·제도에 자유롭게 질문·도전 |
| Mashing Up (어울림) | 다양한 배경·생각·기술의 융합 |
| Risk Taking (위험 감수) | 실패를 두려워하지 않고 시도하는 용기 |
| Mission Orientation (목표 지향) | 명확한 목적을 향한 집중력 |
| Tenacity (끈질김) | 쉽게 포기하지 않는 근성 |
| Learning from Failure | 반복 실험을 통한 성장 |

### 후츠파 × 기업가정신 연결

| 후츠파 | → | 기업가정신 |
|--------|---|---------|
| 질문 | → | 창조적 파괴 (슘페터의 신결합) |
| 위험 감수 | → | 린 스타트업 (MVP + Fail Fast) |
| 어울림 | → | 융합적 사고 (새로운 비즈니스 모델 창조) |
