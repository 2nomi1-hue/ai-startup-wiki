# 스타트업 특성 — 쪽지시험 예상문제 & 정답

---

## 객관식

### Q1. 다음 중 스타트업의 5가지 핵심 특성이 **아닌** 것은?

```
① 문제 해결성 (Problem Solving)
② 파괴성 (Disruptiveness)
③ 수익 안정성 (Revenue Stability)
④ 불확실성 (Uncertainty)
```

**정답: ③ 수익 안정성 (Revenue Stability)**

> 스타트업의 5가지 핵심 특성은 문제 해결성 / 파괴성 / 창조성 / 빠른 성장성 / 불확실성이다.
> 스타트업은 오히려 수익이 불안정한 고위험 구조다.

---

### Q2. CB Insights 통계에 따르면 스타트업 실패 원인 1위는?

```
① 자금 부족
② 팀 문제
③ 시장 수요 부족
④ 기술력 부족
```

**정답: ③ 시장 수요 부족 (42%)**

> 좋은 기술이 있어도 고객이 원하지 않으면 실패한다.
> 이것이 문제 해결성이 5가지 특성 중 1번인 이유다.

---

### Q3. OX 문제 — 맞으면 O, 틀리면 X

```
( O ) 스타트업의 90%는 실패한다.
( X ) 성공한 유니콘 기업의 대부분은 처음부터 비즈니스 모델이 완벽했다.
( O ) 현대 혁신의 80% 이상은 기존 기술의 재조합에서 발생한다.
( O ) 시장은 비타민보다 진통제에 더 기꺼이 지갑을 연다.
```

> X 해설: 실제로 유니콘의 70% 이상이 초기 비즈니스 모델을 전면 피벗(Pivot)한 경험이 있다.

## 서술형

### Q6. 스타트업의 '창조성'이 "무에서 유를 창조하는 것"이 아닌 이유를 설명하고, 실제 기업 사례 1개를 들어 설명하시오. (3줄 이내)

**정답:**

스타트업의 창조성은 '무에서 유'를 만드는 것이 아니라, **기존에 존재하던 기술과 자원을 낯선 방식으로 연결해 새로운 가치를 창출**하는 것이다. (W. Brian Arthur — 현대 혁신의 80% 이상은 재조합)

**사례**: Dyson은 제재소의 '산업용 사이클론 원심력 기술'을 가정용 청소기에 연결해, 100년 된 먼지봉투 청소기 문제를 단숨에 해결하고 영국 시장 점유율 1위를 달성했다.

### Q7. 스타트업과 일반 기업의 차이를 목표 / 자금조달 / 방법론 3가지 측면에서 비교하시오.

**정답:**

| 항목 | 스타트업 | 일반 기업 |
|------|---------|---------|
| 목표 | 지수적 성장 (Exponential) | 안정적 성장 (Stable Growth) |
| 자금조달 | VC / 엔젤 투자 | 은행 대출 / 자체 수익 |
| 방법론 | 실험과 반복 (Iteration) | 계획과 실행 (Execution) |

### Q8. 빠른 성장성(Rapid Growth)을 가능하게 하는 주요 요소 3가지를 쓰고, 각각 한 줄씩 설명하시오.

**정답:**

| 요소 | 설명 |
|------|------|
| **소프트웨어 기반** | 한계비용이 0에 가까워 사용자가 늘어도 추가 비용이 거의 없음. 글로벌 즉시 확장 가능 |
| **거대한 TAM** | 목표 고객군이 방대하고 지리적 제약이 없어 지속 성장 여유가 큼 |
| **네트워크 효과** | 사용자가 늘수록 서비스 가치가 올라가는 구조 (V ∝ N², Metcalfe's Law) |

> 보너스: ChatGPT는 출시 2개월 만에 MAU 1억 명 돌파 — IT 역사상 가장 빠른 지수적 성장 기록

---

### Q9. "스타트업은 실행(Execute)하는 조직이 아니라 탐색(Search)하는 조직이다"라는 말의 의미를 불확실성(Uncertainty)과 연결해서 설명하시오.

**정답:**

일반 기업은 이미 검증된 비즈니스 모델을 '실행'하면 되지만, 스타트업은 **고객이 무엇을 원하는지조차 모르는 극심한 불확실성 속에서 비즈니스 모델을 끊임없이 '탐색'** 해야 한다.

따라서 처음 세운 계획은 거의 반드시 틀리며, 스타트업의 핵심 역량은 **틀린 가설을 빨리 발견하고 수정하는 학습 속도**다. 이것이 MVP 실험과 피벗(Pivot)이 중요한 이유다.

> "어떤 비즈니스 플랜도 고객과 처음 만나기 전까지는 살아남지 못한다." — Steve Blank
