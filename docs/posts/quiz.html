객관식

Q1. 스타트업의 5가지 핵심 특성이 아닌 것은?

① 문제 해결성 (Problem Solving)
② 파괴성 (Disruptiveness)
③ 수익 안정성 (Revenue Stability)
④ 불확실성 (Uncertainty)
정답: ③ 수익 안정성 (Revenue Stability)
스타트업의 5가지 핵심 특성은 문제 해결성 / 파괴성 / 창조성 / 빠른 성장성 / 불확실성이다. 스타트업은 오히려 수익이 불안정한 고위험 구조다.

Q2. CB Insights 통계에 따르면 스타트업 실패 원인 1위는?

① 자금 부족
② 팀 문제
③ 시장 수요 부족
④ 기술력 부족
정답: ③ 시장 수요 부족 (42%)
좋은 기술이 있어도 고객이 원하지 않으면 실패한다.

Q3. OX 문제

( O ) 스타트업의 90%는 실패한다.
( X ) 성공한 유니콘 기업의 대부분은 처음부터 비즈니스 모델이 완벽했다.
( O ) 현대 혁신의 80% 이상은 기존 기술의 재조합에서 발생한다.
( O ) 시장은 비타민보다 진통제에 더 기꺼이 지갑을 연다.
X 해설: 실제로 유니콘의 70% 이상이 초기 비즈니스 모델을 전면 피벗(Pivot)한 경험이 있다.

Q9. "스타트업은 탐색(Search)하는 조직이다"를 불확실성과 연결해 설명하시오.

일반 기업은 검증된 모델을 '실행'하면 되지만, 스타트업은 고객이 무엇을 원하는지조차 모르는 극심한 불확실성 속에서 비즈니스 모델을 끊임없이 '탐색'해야 한다. 처음 세운 계획은 거의 반드시 틀리며, 틀린 가설을 빨리 발견하고 수정하는 학습 속도가 핵심 역량이다. 이것이 MVP 실험과 피벗이 중요한 이유다.

"어떤 비즈니스 플랜도 고객과 처음 만나기 전까지는 살아남지 못한다." — Steve Blank
