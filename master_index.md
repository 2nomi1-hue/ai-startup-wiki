## 🛠️ 운영 규칙

### RAW 폴더에 추가할 때
1. 원본 파일을 `RAW/` 폴더에 그대로 저장
2. 파일명은 날짜 또는 출처 포함 (예: `0319_수업노트.md`, `논문_린스타트업.md`)
3. 이 master_index.md의 RAW 파일 목록에 추가

### WIKI 업데이트 요청 방법
> Claude Code에게 이렇게 말하면 됩니다:
> "RAW 폴더의 [파일명]을 분석해서 WIKI에 정리해줘"
> "새로운 자료가 RAW에 추가됐으니 관련 WIKI 페이지 업데이트해줘"
> "master_index 최신화해줘"

### AI가 이 시스템을 사용하는 방법
1. `master_index.md`를 먼저 읽어 전체 구조 파악
2. 키워드 → WIKI 연결표로 관련 문서 찾기
3. WIKI에 없는 내용은 RAW에서 직접 검색
4. 새로운 인사이트는 WIKI에 추가하고 master_index 업데이트
